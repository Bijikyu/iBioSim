There are versions of iBioSim available for Windows, Linux, and MacOS. You can download the appropriate installation file from:

3  Installation on Linux

Download iBioSim-〈version〉-Linux-x86-Install. Change permissions on this file to allow it to be executed:
chmod u+x iBioSim-〈version〉-Linux-x86-Install. Executing this file will now start the install shield.
